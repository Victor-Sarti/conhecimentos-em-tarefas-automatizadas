Neste laboratório aprendi como construir uma arquitetura serverless baseada em eventos utilizando serviços da AWS.

# Amazon S3

Aprendi como funciona o armazenamento de objetos e como eventos podem ser disparados automaticamente após uploads de arquivos.

# AWS Lambda

Entendi como executar código sem gerenciar servidores, utilizando funções acionadas sob demanda.

# DynamoDB

Aprendi conceitos básicos de banco NoSQL e armazenamento de dados processados automaticamente.

# LocalStack

O uso do LocalStack ajudou a simular serviços AWS localmente, permitindo testes sem depender totalmente da nuvem.

# Arquitetura orientada a eventos

O maior aprendizado foi entender como serviços AWS podem se comunicar automaticamente através de eventos, reduzindo processos manuais e aumentando escalabilidade.

Minhas dificuldades foram: Erros de configuração no LocalStack, e no Debug da função lambda mas logo consegui resolver!
